# Unity-Boilerplate

### A boilerplate to any Unity game projects

# Get Started
#### Download ZIP

Download it - [Unity-Boilerplate-master.zip](https://github.com/Unity-Brazil/Unity-Boilerplate/archive/master.zip)

#### Clone

```sh
$ git clone git@github.com:Unity-Brazil/Unity-Boilerplate.git YourGameFolder
```

## Requirements
* [Unity 5.0.1f1](http://unity3d.com/get-unity) or latest

## Features
* Logo Intro
* Menu
* Cast
* Select Scenes (TODO)
* Options (TODO)
* Quit
* GameManager
* AudioManager
* Pause System
* Mini State Machine

## Folder Structure
```
|-- Resources
  |-- Animations
    |-- clips
    |-- controllers
  |-- Audio
  |-- Fonts
  |-- Images
    |-- gui
    |-- textures
  |-- Materials
  |-- Models
  |-- Prefabs
|-- Scenes
|-- Script
  |-- Controllers
  |-- GameManager
  |-- GUI
```
